创建博客

1、创建仓库
在GitHub上创建一个“username.github.io”为名的仓库，便于博客访问：http://username.github.io。

注意：
每一个github账户最多只能创建一个username.github.io这样可以直接使用域名访问的仓库。

2、配置SSH Key
目的：
方便提交代码到GitHub，使用ssh key来解决本地和服务的连接问题，既可以避免每次都用用户名密码，又可以避免直接使用用户名密码的安全问题。

为啥不安全？
http://chensd.com/2016-08/Safe-username-password-Login-System.html

配置命令：
cd ~/.ssh //检查本机已经存在的ssh密钥

(1).如果：No such file or directory 说明你是第一次使用git。
执行：
ssh-keygen -t rsa -C “邮件地址”

PreferredAuthentications publickey
